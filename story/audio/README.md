# 音频文件使用说明

## 如何添加真人朗读的音频

1. **录制或下载音频文件**
   - 使用在线TTS生成：https://ai.baidu.com/tech/speech
   - 或自己录制真人朗读

2. **文件命名规则**
   将音频文件放在此文件夹，命名为：
   ```
   bedtime_1.mp3  - 小兔子乖乖
   bedtime_2.mp3  - 三只小猪
   bedtime_3.mp3  - 小红帽
   ...
   fable_1.mp3    - 守株待兔
   fable_2.mp3    - 狐假虎威
   ...
   ```

3. **支持的格式**
   - .mp3（推荐）
   - .wav
   - .m4a
   - .ogg

4. **使用方式**
   程序会自动检测音频文件：
   - 如果找到音频文件，播放真人录音
   - 如果没有，使用浏览器TTS朗读

## 音频录制建议

1. **语速**：稍慢，给孩子足够理解时间
2. **停顿**：在段落和句子间适当停顿
3. **语气**：温和、亲切，像讲故事一样
4. **音质**：清晰、无杂音
