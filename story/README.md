# 故事启蒙 - Stories

## 模块概述

睡前故事和寓言故事，培养想象力和语言理解能力。

## 功能列表

### 睡前故事（10个）

1. **小兔子乖乖** - 安全教育
2. **三只小猪** - 勤劳智慧
3. **小红帽** - 安全意识
4. **龟兔赛跑** - 坚持不懈
5. **丑小鸭** - 自信成长
6. **卖火柴的小女孩** - 珍惜幸福
7. **白雪公主** - 善良勇敢
8. **小马过河** - 勇于尝试
9. **拔苗助长** - 遵循规律
10. **坐井观天** - 开阔眼界

### 寓言故事（10个）

1. **守株待兔** - 不要侥幸
2. **狐假虎威** - 认清本质
3. **亡羊补牢** - 及时改正
4. **刻舟求剑** - 因地制宜
5. **掩耳盗铃** - 不要自欺
6. **画蛇添足** - 适可而止
7. **井底之蛙** - 见多识广
8. **盲人摸象** - 全面看问题
9. **农夫与蛇** - 分清善恶
10. **狼来了** - 诚实守信

## 功能设计

### 故事展示
- 故事标题
- 配图/插图
- 文字内容（分段显示）
- 阅读进度

### 互动功能
- 🔊 语音朗读（使用 Web Speech API）
- ⏸️ 暂停/继续
- ⬅️➡️ 上一篇/下一篇
- 🌙 夜间模式
- 📖 字体大小调整

### 故事格式

每篇故事包含：
- 标题
- 寓意/教育意义
- 正文（200-500字）
- 配图（3-5张）
