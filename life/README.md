# 生活启蒙 - Life Skills

## 模块概述

家庭成员、身体部位、日常用品等生活常识，培养生活自理能力。

## 功能列表

### 家庭成员（10种）

1. 爸爸 👨 - Father
2. 妈妈 👩 - Mother
3. 爷爷 👴 - Grandpa (paternal)
4. 奶奶 👵 - Grandma (paternal)
5. 外公 👴 - Grandpa (maternal)
6. 外婆 👵 - Grandma (maternal)
7. 哥哥 👦 - Older brother
8. 姐姐 👧 - Older sister
9. 弟弟 👶 - Younger brother
10. 妹妹 👶 - Younger sister

### 身体部位（15种）

**五官**：眼睛👁️、鼻子👃、嘴巴👄、耳朵👂、舌头👅

**四肢**：手✋、手指👆、脚🦶、脚趾

**其他**：头🗣️、头发💇、肩膀、肚子、膝盖、后背

### 日常用品（25种）

**衣物**：衣服👕、裤子👖、鞋子👟、袜子、帽子🧢、外套

**餐具**：碗🥣、筷子、勺子、叉子、杯子🥛

**厨具**：锅、铲子、刀、案板

**学习用品**：书包🎒、书本📖、铅笔✏️、橡皮、尺子📏

**洗漱用品**：牙刷🪥、牙膏、毛巾、肥皂、梳子

**玩具**：球⚽、积木、娃娃🎎、拼图

## 功能设计
