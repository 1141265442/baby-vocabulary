# 字母启蒙 - Alphabet

## 模块概述

学习26个英文字母，包括大小写、发音和以该字母开头的单词。

## 功能列表

### 26个字母

Aa Bb Cc Dd Ee Ff Gg Hh Ii Jj Kk Ll Mm
Nn Oo Pp Qq Rr Ss Tt Uu Vv Ww Xx Yy Zz

### 每个字母包含

- 大写字母
- 小写字母
- 发音（phonics）
- 以该字母开头的单词（3-5个）
- 单词配图

### 单词示例

A - Apple（苹果）🍎、Ant（蚂蚁）🐜、Airplane（飞机）✈️
B - Ball（球）⚽、Banana（香蕉）🍌、Bear（熊）🐻
C - Cat（猫）🐱、Car（汽车）🚗、Cake（蛋糕）🍰
...

## 功能设计

### 学习模式
1. **卡片模式**：单独展示每个字母
2. **列表模式**：A-Z 滚动浏览
3. **歌曲模式**：ABC 字母歌

### 互动功能
- 🔊 字母发音
- 🔊 单词发音
- 🎨 配图展示
- ⬅️➡️ 左右切换
- 🔀 随机模式

## 页面结构

```
alphabet/

```
