# 字母启蒙 - Alphabet

## 模块概述

通过游戏化互动方式，帮助幼儿学习26个英文字母的大小写、发音和相关单词。

## 核心设计理念

**游戏化学习**：将字母学习融入有趣的小游戏，让孩子在玩中学
**多感官互动**：视觉（配图）、听觉（发音）、触觉（点击操作）
**循序渐进**：从认识字母到拼写单词，难度逐步提升
**即时反馈**：正确/错误及时反馈，增强学习成就感

## 游戏模式

### 🎮 模式一：字母卡片（学习模式）

**玩法**：
- 26个字母卡片，左右滑动切换
- 点击卡片朗读字母发音
- 显示大写、小写字母
- 展示3个以该字母开头的单词配图
- 点击单词朗读发音

**内容**：
```
Aa - 🍎 Apple、🐜 Ant、✈️ Airplane
Bb - ⚽ Ball、🍌 Banana、🐻 Bear
Cc - 🐱 Cat、🚗 Car、🍰 Cake
...
```

### 🎮 模式二：字母配对（记忆游戏）

**玩法**：
- 屏幕显示6-12张卡片（3-6对字母）
- 卡片初始背面朝上
- 点击翻转卡片，找出相同字母的大小写
- 配对成功卡片消失，配对失败卡片翻回
- 全部配对完成胜利

**难度**：
- 简单：3对（Aa, Bb, Cc）
- 中等：5对（Aa-Ee）
- 困难：10对（Aa-Jj）

### 🎮 模式三：气球大爆炸

**玩法**：
- 屏幕底部飘出气球，每个气球上有字母
- 语音播放目标字母，孩子点击正确气球
- 点对气球爆炸🎉，点错气球摇晃❌
- 限时60秒，看能炸掉多少个气球

**变种**：
- 听音点字母：播放字母发音，点击对应字母
- 大小写配对：点击大写字母后再点击对应小写
- 单词首字母：播放单词"Apple"，点击字母"A"

### 🎮 模式四：字母拖拽（拼写游戏）

**玩法**：
- 显示图片（如🍎）
- 底部散落字母卡片
- 孩子将字母拖到正确位置拼出单词
- 拼对播放单词发音和庆祝动画

**内容**：
```
🍎 → A - P - P - L - E
🐱 → C - A - T
🐕 → D - O - G
🚗 → C - A - R
```

### 🎮 模式五：字母连线

**玩法**：
- 左边是大写字母，右边是小写字母
- 用线将对应的大小写字母连接
- 全部连对后播放庆祝动画

### 🎮 模式六：字母歌（唱歌模式）

**功能**：
- 播放ABC字母歌音频
- 配合动画，字母随节奏跳动
- 歌词高亮显示
- 可调整播放速度

### 游戏界面元素

- **顶部导航**：返回按钮、当前模式标题
- **游戏区域**：中央互动区域
- **状态显示**：得分/进度/计时器
- **底部控制**：提示按钮、重新开始
- **反馈动画**：
  - ✅ 正确：绿色勾+星星动画

## 技术实现

### 语音功能
- 使用 Web Speech API
- 字母发音：`lang='en'`
- 单词朗读：英文+中文翻译

### 动画效果
- CSS 动画：气球飘动、卡片翻转
- Canvas 连线：字母连线模式
- 粒子效果：胜利庆祝

### 存储功能
- localStorage 保存学习进度
- 记录已学字母、游戏得分
