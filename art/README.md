# 艺术启蒙 - Art

## 模块概述

在线涂色、简笔画教学，培养创造力和动手能力。

## 功能列表

### 在线涂色

**主题分类**（30张）

**动物类**（10张）
- 小猫🐱、小狗🐶、兔子🐰、小熊🐻、小鸟🐦
- 鱼🐟、蝴蝶🦋、蜜蜂🐝、青蛙🐸、乌龟🐢

**植物类**（10张）
- 苹果🍎、花朵🌸、树木🌳、蘑菇🍄、仙人掌🌵
- 向日葵🌻、玫瑰🌹、竹子🎋、荷叶、松树

**其他类**（10张）
- 房子🏠、汽车🚗、飞机✈️、船⛵、太阳☀️
- 月亮🌙、星星⭐、气球🎈、蛋糕🎂、礼物🎁

### 简笔画教学

**基础图形**（10种）
- 圆形、方形、三角形、直线、曲线
- 波浪线、螺旋线、心形、星形、月亮

**简单绘画**（20种）
- 水果：苹果🍎、香蕉🍌、葡萄🍇
- 动物：小鸡🐤、小鱼🐟、蝴蝶🦋
- 植物：小花🌸、小树🌱、太阳☀️
- 物品：房子🏠、汽车🚗、杯子🥛

## 功能设计

### 在线涂色
- 线稿展示
- 颜色选择器（12-20种颜色）
- 点击区域填充颜色
- 清空/重置
- 保存作品（下载图片）

### 简笔画教学
- 分步骤教学（3-5步）
- 动画演示
- 跟着画（描红）
- 自由画板

## 技术要点

- 使用 Canvas 或 SVG 实现涂色
- 触摸/鼠标事件处理
- 图片导出功能

## 页面结构

```
art/
├── index.html        # 主页面
├── coloring.html     # 在线涂色
└── drawing.html      # 简笔画
```
