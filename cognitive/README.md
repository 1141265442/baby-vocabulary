# 认知启蒙 - Cognition

## 模块概述

颜色、动物、水果蔬菜、交通工具等基础认知，培养观察力和记忆力。

## 功能列表

### 颜色认知（10种）

1. 红 🔴 - Red
2. 橙 🟠 - Orange
3. 黄 🟡 - Yellow
4. 绿 🟢 - Green
5. 蓝 🔵 - Blue
6. 紫 🟣 - Purple
7. 黑 ⚫ - Black
8. 白 ⚪ - White
9. 灰 ⬜ - Gray
10. 棕 🟤 - Brown

### 动物世界（30种）

**农场动物**：鸡🐔、鸭🦆、牛🐮、羊🐑、猪🐷、马🐴

**森林动物**：老虎🐯、狮子🦁、大象🐘、猴子🐵、熊猫🐼、长颈鹿🦒

**海洋动物**：鱼🐟、海豚🐬、鲸鱼🐳、螃蟹🦀、海星⭐、章鱼🐙

**昆虫类**：蝴蝶🦋、蜜蜂🐝、蜻蜓、蚂蚁🐜、瓢虫🐞

**鸟类**：小鸟🐦、孔雀🦚、鹦鹉、猫头鹰🦉、企鹅🐧

**宠物**：猫🐱、狗🐶、兔子🐰、仓鼠🐹

### 水果蔬菜（20种）

**水果**：苹果🍎、香蕉🍌、橙子🍊、草莓🍓、葡萄🍇、西瓜🍉、樱桃🍒、桃子🍑、梨🍐、菠萝🍍

**蔬菜**：白菜🥬、胡萝卜🥕、西红柿🍅、黄瓜🥒、南瓜🎃、土豆🥔、玉米🌽、茄子🍆、青椒🫑、萝卜

### 交通工具（15种）

**陆地**：汽车🚗、公交车🚌、火车🚂、自行车🚲、摩托车🏍️

**空中**：飞机✈️、直升机🚁、火箭🚀

**水上**：船🚢、轮船🚢、帆船⛵

**特殊**：消防车🚒、警车🚓、救护车🚑
