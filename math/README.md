# 数学启蒙 - Math

## 模块概述

数字认知、简单运算和几何形状认知，培养数学思维。

## 功能列表

### 认数字（1-100）

**阶段一：1-10**
- 数字认知
- 数数练习
- 数量对应

**阶段二：11-20**
- 数字认知
- 数位理解

**阶段三：1-100**
- 数字认知
- 规律发现

### 简单运算

**加法（10以内）**
- 1+1=2、2+3=5...
- 图形化演示

**减法（10以内）**
- 5-2=3、8-3=5...
- 图形化演示

### 形状认知

**基本形状（6种）**
- 圆形 ⭕
- 方形/正方形 ⬜
- 三角形 🔺
- 长方形 ▬
- 椭圆形 ⬭
- 星形 ⭐

**生活中的形状**
- 太阳是圆形
- 门是长方形
- 三角尺是三角形

## 功能设计

### 认数字页面
- 数字卡片展示
- 数量对应（如：3 🍎🍎🍎）
- 数数动画
- 随机/顺序切换

### 简单运算页面
- 题目展示
- 答案选择（多选）
- 对错反馈
- 积分统计

### 形状认知页面
- 形状卡片
- 形状配对游戏
- 生活中的形状找一找

## 页面结构

```
math/
├── index.html        # 主页面
├── numbers.html      # 认数字
├── arithmetic.html   # 简单运算
└── shapes.html       # 形状认知
```
