  方法：Netlify Drop                                                                                                                       
  
  1. 打开浏览器访问：https://app.netlify.com/drop
  2. 将 英文 文件夹（或 vocabulary.html 文件）直接拖拽到页面上
  3. 等待几秒，会自动生成一个网址，例如：https://xxx.netlify.app

  如果要用 GitHub Pages：

  1. 在 GitHub 创建新仓库（命名为 baby-vocabulary）
  2. 运行以下命令：

  cd "C:\Users\harlan\Desktop\baby"
  git remote add origin https://github.com/你的用户名/baby-vocabulary.git
  git branch -M main
  git push -u origin main

  3. 在仓库 Settings → Pages → Source 选择 main 分支
