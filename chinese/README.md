# 汉字启蒙 - Chinese Characters

## 模块概述

学习基础汉字笔画和简单汉字，培养汉字认知能力。

## 功能列表

### 基本笔画（8种）

| 笔画 | 名称 | 例字 |
|------|------|------|
| 一 | 横 | 一、二、三 |
| 丨 | 竖 | 十、中、王 |
| 丿 | 撇 | 八、人、入 |
| 乀 | 捺 | 八、人、大 |
| 丶 | 点 | 主、头、火 |
| 亅 | 钩 | 小、水、手 |
| 乚 | 弯钩 | 子、了、心 |
| 乛 | 横折 | 口、日、田 |

### 简单汉字（50个）

**数字类**：一、二、三、四、五、六、七、八、九、十

**自然类**：日、月、星、云、雨、雪、风、火、水、土、山、石

**人物类**：人、大、小、女、子、口、目、耳、手、足

**动物类**：牛、羊、马、鸟、鱼、虫、犬

**植物类**：木、禾、米、竹、草

**其他类**：上、下、左、右、中、天、地、心、力

## 功能设计

### 笔画教学
- 笔画动画演示
- 笔顺展示
- 描红练习

### 汉字学习
- 卡片展示（拼音、笔画数、例词）
- 动画演示笔顺
- 语音朗读

### 互动游戏
- 笔画连连看
- 汉字拼图
- 听音选字
