# 儿歌音频下载指南

## 方式一：PowerShell 自动下载（推荐）

在 Windows 上运行以下命令下载免费儿歌：

```powershell
# 以管理员身份运行 PowerShell，然后执行：
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
cd "C:\Users\harlan\Desktop\baby\music"
.\download_songs.ps1
```

## 方式二：手动下载

### 免费儿歌资源网站

1. **Internet Archive**（公共领域，免费）
   - 访问：https://archive.org/details/ChildrensMusic
   - 访问：https://archive.org/search.php?query=nursery+rhymes
   - 搜索歌名后点击 "MP3" 下载

2. **Free Music Archive**
   - 访问：https://freemusicarchive.org
   - 搜索 "children" 或 "nursery rhyme"

3. **Jamendo Music**
   - 访问：https://www.jamendo.com
   - 搜索儿童音乐

### 中文儿歌下载

1. **喜马拉雅**
   - https://www.ximalaya.com
   - 搜索儿歌，可免费收听部分

2. **腾讯视频/爱奇艺**
   - 搜索儿歌，使用下载工具提取音频

3. **抖音/快手**
   - 收藏后使用提取工具

## 方式三：手机 App 推荐

- 喜马拉雅儿童版
- 凯叔讲故事
- 宝宝巴士
- 小伴龙

## 音频文件存放

下载后将 MP3 文件放到 `audio` 文件夹，命名为：
- 中文：`两只老虎.mp3`
- 英文：`Twinkle_Twinkle_Little_Star.mp3`
