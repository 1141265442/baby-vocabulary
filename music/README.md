## 功能设计

### 页面元素
- 🎵 音频播放器（播放/暂停/进度）
- 📝 歌词显示（滚动高亮）
- 🎨 配图/动画背景
- 🔁 循环播放开关
- ⏮️⏭️ 上一首/下一首
- 📋 歌曲列表（点击切换）

### 技术要点
- 使用 HTML5 Audio API
- 移动端音频自动播放限制处理
- 歌词同步高亮
- 本地音频或在线音频源
